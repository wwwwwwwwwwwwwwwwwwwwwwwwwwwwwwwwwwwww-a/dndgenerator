# Государственное бюджетное общеобразовательное учреждение города Москвы «Школа №1561»

Наш проект посвящен разработке веб-сервиса для генерации игровых и неигровых персонажей для настольной ролевой игры Dungeons & Dragons. Сервис решает проблему длительного и сложного создания персонажей, с которой сталкиваются новички и неопытные мастера при подготовке и проведении своих игровых сессий.
Мы создали веб-приложение на Python с использованием фреймворка Flask и базы данных SQLite3. Генерация персонажей основана на правилах D&D 5e. Нейросеть создает персонажа и подбирает для него характеристики и пишет структурированную предысторию. Реализованы регистрация пользователей, сохранение персонажей и детальная персонализация интерфейса (тема, шрифт). 
У нашего проекта есть аналоги, которые не имеют системы аккаунтов, но у некоторые предлагают возможность экспорта персонажа в pdf или генерацию внешности персонажа.
В дальнейшем проект можно улучшить, реализовав экспорт персонажа в pdf формат для удобства печати или добавить генерации внешности персонажей картинкой.
